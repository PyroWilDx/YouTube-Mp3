# YouTube-Mp3

[**YouTube-Mp3**](https://github.com/PyroWilDx/YouTube-Mp3/) is a tool designed to download Mp3 audio from YouTube videos. This tool also embeds a high-quality thumbnail (achieved through image upscaling) or a custom image into the audio file.

### How To Use

- Install all required Python packages w/ [```Requirements.txt```](./Requirements.txt).

```
pip install -r Requirements.txt
```

- Run [```src/Main.py```](src/Main.py) w/ Python.
